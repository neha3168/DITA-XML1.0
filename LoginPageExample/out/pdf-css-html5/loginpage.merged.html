How to Login?

Login using password

This topic describes the process for the happy path(straight-forward login), by using correct password and providing correct captcha.

The end-user must know the password for the login.

  1. Enter the login/ UserId for the Login, This is generally the email ID you have used to create your account.
  2. Enter the password that was set while creating the account.
  3. Enter the capctha that appears in the Captcha box.
  4. Press Enter Key or click on Login button
The User will be successfully logged into the portal if the UserId, Password and captcha are correct.

How to Reset Password?

Forgot Password

Here we define how to reset user password, if they forget their password.

The user must have the active email Id or phone number that was provided while creating their account, incase they forgot their password.

  1. Click on the forgot password link provided on the login page.
  2. A new page will open, containing a textbox to enter the emailId or phone number for password recovery.
  3. Enter the email Id or the phone number and click on the button "Reset Password".
  4. An email containing the link to reset password, will be sent in the email provided, if it is the same email that was given at the time of account creation. Otherwise, an error message that the associated email is incorrect will appear
  5. If a phone number was provided at the time of account creation for resetting the password. Then a link will be directly sent as a message on the phone number. The password can be reset by clicking on the link.
The users should be able to reset their password successfully upon clicking the link received in their email or phone number.

How to Reset Captcha?

Complex Captcha Reset

Here we define how to reset captcha incase the user finds it complex to decode.

Inorder to reset the captch if the current image seems complex, press on the reset button right next to the Captcha box
